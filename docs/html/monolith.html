Product

상품 관리 API

상품 API 가이드

상품 생성

새로운 상품을 생성합니다.

비즈니스 규칙

  • SKU는 중복될 수 없습니다 (자동 생성)
  • 가격은 0.01원 이상 99,999,999.99원 이하여야 합니다
  • 초기 재고 수량은 0 이상이어야 합니다
  • 카테고리 ID는 정의된 값 중 하나여야 합니다 (1-10)

아키텍처

이 API는 헥사고날 아키텍처의 인바운드 어댑터를 통해 구현되었습니다.

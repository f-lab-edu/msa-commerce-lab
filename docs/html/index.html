📋 프로젝트 개요
이 프로젝트는 현대적인 마이크로서비스 아키텍처와 헥사고날 아키텍처 패턴을 활용하여 확장 가능한 커머스 플랫폼을 구축하는 것을 목표로 합니다.
📦 Monolith API
상품 관리 API를 포함한 모놀리식 서비스 문서입니다. 상품 생성, 조회, 수정, 삭제 기능과 카테고리 관리 기능을 제공합니다.
